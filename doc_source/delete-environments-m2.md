# Delete a AWS Mainframe Modernization runtime environment<a name="delete-environments-m2"></a>

Use the AWS Mainframe Modernization console  to delete a AWS Mainframe Modernization runtime environment\.

These instructions assume that you have completed the steps in [Setting up AWS Mainframe Modernization](setting-up.md)\.

## Delete a runtime environment<a name="delete-environments-m2.console"></a>

If you need to delete a AWS Mainframe Modernization runtime environment, make sure to delete any deployed applications from the environment first\. You cannot delete a runtime environment that has deployed applications\.

**To delete an environment**

1. Open the AWS Mainframe Modernization console at [https://console\.aws\.amazon\.com/m2/](https://console.aws.amazon.com/m2/)\.

1. In the region selector, choose the AWS Region where the environment you want to delete was created\.

1. On the **Environments** page, choose the environment you want to delete, then choose **Actions** and **Delete environment**\.

1. In **Delete environment**, enter `delete` to confirm that you want to delete the runtime environment, then choose **Delete**\.
